# 学术写作规范

## 功能概述

本技能提供法学学术写作的专业规范支持，覆盖以下能力：

1. **论文结构规范**：标准法学论文结构设计
2. **引用格式**：GB/T 7714 格式及法学常用引用规范
3. **学术语言**：学术写作的语言风格和表达规范
4. **图表规范**：学术图表的设计与排版标准
5. **参考文献管理**：参考文献的整理与编排

## 核心规范

### 论文结构
- 标题 → 摘要 → 关键词 → 引言 → 文献综述 → 研究方法 → 分析 → 讨论 → 结论 → 参考文献 → 附录

### 引用格式（GB/T 7714）
- **期刊文章**：作者. 题名[J]. 刊名, 年, 卷(期): 起止页码.
- **专著**：作者. 书名[M]. 出版地: 出版社, 出版年.
- **学位论文**：作者. 题名[D]. 出版地: 出版单位, 年份.
- **网络资源**：作者. 题名[EB/OL]. (发布日期)[引用日期]. URL.

### 学术语言规范
- 客观陈述，避免主观判断
- 使用学术术语，定义清晰
- 逻辑推理严密，论证充分
- 段落结构完整，过渡自然
- 避免口语化表达

### 表格规范
- 表序和表题置于表上方
- 三线表格式
- 数据对齐和单位标注

### 插图规范
- 图序和图题置于图下方
- 坐标轴标注完整
- 图例清晰

## 输出格式

- 格式检查报告
- 引用格式修正建议
- 语言润色建议
- 完整格式化文档
