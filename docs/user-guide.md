# 法学社会调查多智能体系统 - 用户指南

## 系统概述
本系统是一个覆盖法学社会调查全流程的 WorkBuddy 专家包，包含7个阶段智能体和5个专业能力模块。

## 阶段流程
1. **选题沟通** → 2. **方案确定** → 3. **问卷设计** → 4. **问卷方案** → 5. **结果统计** → 6. **数据分析** → 7. **报告输出**

## 使用方法
- 在 WorkBuddy 中输入 `/law-survey start` 启动流程
- 或直接激活对应阶段的自动化任务
- 每阶段产出经审核后提供给下一阶段

## 系统架构
- **Workflow层**：7阶段顺序编排，支持回退和跳过
- **Agent层**：8个专业化智能体（含总控编排器）
- **Skill层**：5个可复用能力模块
