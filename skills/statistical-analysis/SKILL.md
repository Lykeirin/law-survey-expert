# 统计分析方法库

## 功能概述

本技能提供法学实证研究所需的统计分析方法支持，覆盖以下能力：

1. **描述性统计**：频数、集中趋势、离散程度、分布形态
2. **推断性统计**：参数估计、假设检验
3. **相关与回归分析**：相关分析、线性回归、Logistic回归
4. **因子分析**：探索性因子分析、验证性因子分析
5. **差异检验**：t检验、方差分析、卡方检验
6. **中介/调节效应**：中介效应检验、调节效应检验

## 核心方法

### 描述性统计
- 分类变量：频数、百分比
- 连续变量：均值、标准差、中位数、四分位数
- 分布形态：偏度、峰度、正态性检验

### 推断性统计
- 置信区间估计
- 单样本/独立样本/配对样本 t 检验
- 单因素/多因素方差分析
- 卡方独立性检验

### 回归分析
- 简单线性回归
- 多元线性回归
- 二元/多元 Logistic 回归
- 有序 Logistic 回归

### 因子分析
- KMO 检验（≥0.6）
- Bartlett 球形检验（p < 0.05）
- 主成分分析 / 最大似然法
- 方差最大旋转

## 输出格式

- 统计结果表格
- 统计结果解读
- 效果量指标
- 统计图表（配合数据可视化技能）
