# 法学知识检索

## 功能概述

本技能为法学社会调查提供权威知识检索能力。当环境已连接 **元典法律数据（yuandian-mcp）** 时，优先使用其 MCP 工具做检索；未连接时降级为模型记忆 + 软标注。

检索类型与对应工具：

1. **法律条文检索（现行有效法条）** → `mcp__yuandian-mcp__yuandian_law_vector_search`
   - 法条级语义检索，可按 `fatiao_filter` 过滤时效性（现行有效 / 失效 / 已被修改 / 部分失效 / 尚未生效）、一级效力级别（宪法 / 法律 / 司法解释 / 行政法规 / 地方性法规…）、实施日期范围。
   - 返回 `extra.fatiao`，每条含法条内容 + 所属法规信息（不是整部法规全文）。
2. **案例文书检索（裁判语义检索）** → `mcp__yuandian-mcp__yuandian_case_vector_search`
   - 普通案例 + 权威案例（指导性案例 / 公报案例 / 典型案例…）语义相似度检索。
   - 可按 `wenshu_filter` 过滤案件类别、案由、文书种类、法院、地域、结案日期、是否仅权威案例。
   - 返回整理后的完整案例内容 + 相似度评分。
3. **企业 / 主体信息检索** → `mcp__yuandian-mcp__yuandian_rh_*` 系列（企业工商、司法风险、行政处罚、知识产权、股权冻结等），多用于尽职调查类子课题。
4. **制度背景 / 学术脉络**：元典未覆盖时，由模型基于训练知识提供，并软标注 `[⚠ 未经元典校验]`。

## 使用方法（在 WorkBuddy 中，元典 MCP 已连接）

### 法条检索示例
调用 `yuandian_law_vector_search`，`query` 用自然语言问题：
- "夫妻共同财产的现行规定，要法条原文和出处"
- "限制民事行为能力人的行为效力，现行有效条文"
- 可加 `fatiao_filter={"sxx":["现行有效"],"effect1":["法律","司法解释"]}` 缩小到现行有效的法律 / 司法解释。

### 案例检索示例
调用 `yuandian_case_vector_search`，`query` 用自然语言 + 案由：
- "股东转让未届期认缴股权后，对转让前公司债务是否担责？类似裁判"
- 可加 `wenshu_filter={"ay":["股权转让纠纷"],"dianxing":true}` 只看权威案例。

## 输出格式

所有检索结果统一为结构化列表：
- 标题 / 名称（法条：所属法规 + 条号；案例：案由 + 法院 + 案号）
- 来源 / 出处
- 核心内容摘要
- 相关度 / 相似度评分
- 引用格式（GB/T 7714）

## 注意事项

- 法条务必注明发布日期与效力状态；元典返回已含时效性，须原样呈现，不得声称"现行有效"除非 `fatiao_filter` 已限定 `sxx:["现行有效"]`。
- 案例须注明裁判法院与案号；权威案例优先引用。
- 元典未覆盖或调用失败，降级为模型记忆并追加 `[⚠ 未经元典校验]`，正式用于诉讼 / 教学 / 研究前请以官方最新文本复核。
- 企业信息检索（rh_*）注意个人信息保护与数据合规。
