# 问卷设计方法论

## 功能概述

本技能提供法学社会调查问卷设计的专业方法论支持，覆盖以下核心能力：

1. **变量操作化**：将法律概念转化为可测量指标
2. **量表开发**：设计符合法学研究需求的测量量表
3. **信效度检验**：评估问卷和量表的信度和效度
4. **抽样设计**：确定抽样方法、计算样本量

## 核心方法

### 变量操作化
- 概念定义 → 维度分解 → 指标设计
- 法学概念的操作化示例：法律意识、司法信任、权利认知

### 量表类型
- Likert 量表（5点/7点）
- 语义差异量表
- 视觉模拟量表
- Guttman 量表

### 信度检验
- 内部一致性信度：Cronbach's α ≥ 0.7
- 折半信度
- 重测信度

### 效度检验
- 内容效度：专家评审法
- 结构效度：探索性/验证性因子分析
- 效标关联效度

## 输出格式

- 变量操作化表（概念 → 维度 → 指标 → 题目）
- 量表设计说明
- 信效度检验报告
- 抽样方案
