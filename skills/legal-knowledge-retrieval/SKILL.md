# 法学知识检索

## 功能概述

本技能提供法学研究中的知识检索能力，支持以下检索类型：

1. **法律条文检索**：查询现行法律法规、司法解释
2. **学术文献检索**：检索法学核心期刊论文、学位论文
3. **案例检索**：查询指导性案例、典型案例
4. **制度背景检索**：了解特定法律制度的立法背景和实践现状

## 使用方法

### 法律条文检索
```python
# 检索关键词
laws = search_legal_provisions("个人信息保护法 数据权")
```

### 学术文献检索
```python
# 按主题检索
papers = search_academic_papers("数据权利 赋权模式")
```

### 案例检索
```python
# 按案由检索
cases = search_cases("个人信息侵权 损害赔偿")
```

## 输出格式

所有检索结果统一输出为结构化列表，包含：
- 标题/名称
- 来源/出处
- 核心内容摘要
- 相关度标注
- 引用格式（GB/T 7714）

## 注意事项

- 法律条文应注明发布日期和效力状态
- 学术文献优先引用近5年核心期刊
- 案例应注明裁判法院和案号
