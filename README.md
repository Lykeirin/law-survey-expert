# 法学社会调查多智能体系统 (Law Survey Expert)

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)
[![WorkBuddy](https://img.shields.io/badge/WorkBuddy-Plugin-blue)](https://www.codebuddy.cn/docs/workbuddy)

## 概述

一个面向法学研究者的社会调查全流程 AI 辅助系统，覆盖从选题到报告的 **7个阶段**：

① **选题沟通** → ② **方案确定** → ③ **问卷设计** → ④ **问卷方案** → ⑤ **结果统计** → ⑥ **数据分析** → ⑦ **报告输出**

采用 **Workflow × Agent × Skill** 三层架构设计。

## 功能特点

- **全流程覆盖**：从选题沟通到最终报告输出，一站式完成
- **专业化 Agent**：每个阶段由独立智能体负责，具备领域专业知识
- **可复用 Skill**：法学知识检索、问卷设计方法论、统计分析、学术写作等核心能力模块化
- **灵活编排**：支持回退、跳过、从任意阶段开始
- **一键启动**：输入 `/law-survey start` 即可开始

## 系统架构

```
┌─────────────────────────────────────────────────────┐
│  Workflow层（7阶段编排）                              │
│  选题 → 方案 → 问卷 → 实施 → 统计 → 分析 → 报告     │
├─────────────────────────────────────────────────────┤
│  Agent层（8个智能体）                                 │
│  总控编排器 + 7阶段专业智能体                         │
├─────────────────────────────────────────────────────┤
│  Skill层（5个可复用能力模块）                          │
│  法学知识检索 | 问卷设计方法论 | 统计方法 | 学术写作    │
└─────────────────────────────────────────────────────┘
```

## 快速开始

### 前提条件
- 已安装 [WorkBuddy](https://www.codebuddy.cn/)
- 已注册并登录 WorkBuddy 账号

### 安装方法

#### 方法一：从 GitHub 安装
```bash
git clone https://github.com/[用户名]/law-survey-expert.git
ln -sf "$(pwd)/law-survey-expert" ~/.workbuddy/plugins/local/law-survey-expert
```

#### 方法二：一键安装脚本
```bash
bash install.sh
```

#### 方法三：手动安装
1. 下载本仓库 ZIP 包
2. 解压到 `~/.workbuddy/plugins/local/law-survey-expert/`
3. 重启 WorkBuddy

### 使用方式

在 WorkBuddy 中输入以下命令：
```
/law-survey start
```

系统将自动评估你当前所处的阶段，并引导你进入正确的智能体。

## 智能体详细说明

| 智能体 | 阶段 | 核心产出 | 角色 |
|--------|------|---------|------|
| 🏛️ 总控编排器 | 全流程 | 阶段调度 | 入口和调度中心 |
| 📋 选题沟通 | Stage 1 | 《选题论证报告》 | 互动式选题聚焦 |
| 📋 方案确定 | Stage 2 | 《研究方案书》 | 研究设计方法论 |
| 📋 问卷设计 | Stage 3 | 完整问卷+设计说明 | 测量工具开发 |
| 📋 问卷方案 | Stage 4 | 《调查实施方案》 | 调查项目管理 |
| 📊 结果统计 | Stage 5 | 《数据统计报告》 | 统计分析 |
| 📊 数据分析 | Stage 6 | 《数据分析报告》 | 法学实证分析 |
| 📄 报告输出 | Stage 7 | 《法学社会调查报告》 | 学术写作 |

## 技能模块

- **法学知识检索**：检索法律条文、司法解释、学术文献
- **问卷设计方法论**：变量操作化、量表开发、信效度检验
- **统计分析方法库**：描述统计、推断统计、回归分析
- **学术写作规范**：论文结构、GB/T 7714 引用格式
- **数据可视化**：图表选择、设计原则、报告排版
